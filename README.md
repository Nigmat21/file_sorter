# File Sorter

Простой скрипт на Python для сортировки файлов в папке по типу: изображения и видео.

## Что делает
- Берёт указанную папку на компьютере.
- Создаёт подпапки `Images` и `Videos` (если их нет).
- Перемещает файлы с расширениями:
  - Изображения: `.jpg`, `.jpeg`, `.png`
  - Видео: `.mp4`, `.avi`, `.mov`
- Если файл с таким именем уже есть — добавляет `1` к имени.
- Игнорирует другие типы файлов.

## Технологии
- Python
- Модули: `os`, `shutil`
